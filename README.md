# ✈️ TravelGo — Luxury Travel Booking Platform

A fully responsive, multi-page luxury travel booking web app built with pure **HTML, CSS & JavaScript**. No frameworks, no build tools — just open and deploy.

---

## 🌐 Live Demo
🔗 https://p-diwja.github.io/TravelGo

## 📁 Project Structure

```
travelgo/
│
├── index.html              → Home page
├── dashboard.html          → User dashboard
│
├── pages/
│   ├── property.html       → Property detail page
│   ├── booking.html        → Booking & payment page
│   └── confirm.html        → Booking confirmation
│
├── css/
│   ├── style.css           → Global shared styles
│   ├── home.css            → Home page styles
│   ├── property.css        → Property page styles
│   ├── booking.css         → Booking page styles
│   └── dashboard.css       → Dashboard styles
│
├── js/
│   ├── main.js             → Home page interactions
│   ├── booking.js          → Payment form logic
│   └── dashboard.js        → Dashboard interactions
│
└── README.md
```

---

## 🚀 Features

- 🏠 **Home Page** — Hero search, category filters, destinations, experiences, testimonials
- 🏡 **Property Page** — Photo gallery, amenities, itinerary, reviews, booking sidebar
- 💳 **Booking Page** — 4-step checkout, insurance options, Apple Pay / Card payment
- 📊 **Dashboard** — Trips, rewards points, activity chart, quick actions
- 📱 **Fully Responsive** — Mobile, tablet and desktop
- ⚡ **No Dependencies** — Pure HTML, CSS, JavaScript only

## 🖥️ Run Locally

No installation needed:

```bash
# Clone the repo
git clone https://github.com/YOUR-USERNAME/travelgo.git

# Open index.html in your browser
```

Or use **VS Code Live Server** extension for hot reload.

## 📄 Pages Overview

### 🏠 Home
Hero with search bar, category filters, destinations grid, featured experiences, stats banner (500+ Hotels, 120+ Destinations, 15k Travellers, 4.9 Rating), testimonials, newsletter CTA.

### 🏡 Property Detail
5-photo gallery, property facts, amenities, 3-day itinerary, map, guest reviews with rating bars, local experiences, similar properties, sticky booking sidebar.

### 💳 Booking & Payment
4-step progress tracker, traveler details form, insurance options, Apple Pay / Google Pay / card form with formatting, promo code support, order summary.

### 📊 Dashboard
Sidebar nav, next trip hero card with countdown, Gold Member rewards card, activity bar chart, quick actions, upcoming trips list.
